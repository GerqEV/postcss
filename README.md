PostCSS
==============

- Install (npm, gulp required):
$ npm install gulp-postcss postcss-mixins postcss-simple-vars postcss-nested autoprefixer-core --save-dev

- install plugins:
$ npm install css-mqpacker
$ npm install cssnano


- create gulpfile.js
- create postcss:
- compile (gulp, cli, broccoli, fly, meteor...):
$ gulp css

- Result: 
	- css
	- minified (nano)
